MLU-EXPLAIN

Visual explanations of core machine learning concepts.


Machine Learning University (MLU) is an education initiative designed to teach Amazon's technical community machine learning theory and practical application. We offer a number of courses on different machine learning topics.

As supplementary material, MLU-Explain exists to explain core machine learning concepts using visual essays in a fun, informative, and accessible manner. These essays don't yet cover a large number of the concepts covered in our MLU courses, but we are working to add more. Stay tuned!

The Random Forest Algorithm

How the majority vote and well-placed randomness can enhance the decision tree model.

The Bias-Variance Tradeoff

Learn the tradeoff between under- and over-fitting models, how it relates to bias and variance, and explore interactive examples related to LASSO and KNN.

Double Descent 1

A visual introduction to the double descent phenomena in modern machine learning. Learn what the phenomena is, how it relates to the bias-variance tradeoff, the importance of the interpolation regime, and a theory for what's causing the phenomena. on.

Double Descent 2

A more in-depth explanation of the double descent phenomena. The article builds on the cubic spline example introduced in Double Descent 1, describing in mathematical detail what's occuring during double descent.

Train, Test, & Validation Splits

Learn why best practice recommends to split your dataset into a training set, a testing set, and a validation set, and explore the utility of each with a live machine learning model.
